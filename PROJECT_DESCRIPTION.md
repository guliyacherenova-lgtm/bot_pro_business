# Описание проекта: AI-бот квалификации кандидатов NL International

## 🎯 Цель проекта
Автоматизировать процесс первичного отбора и «прогрева» кандидатов, которые интересуются бизнесом NL International. Бот должен заменить наставника на этапе первого знакомства, собрать данные о пользователе и направить его к регистрации.

## 👥 Целевая аудитория
- Люди, ищущие дополнительный доход.
- Мамы в декрете (особый сегмент).
- Молодые люди от 14 лет, стремящиеся к финансовой независимости.

## 😫 Боли, которые решает автоматизация
- **Рутина**: Наставнику не нужно вручную спрашивать имя, город и возраст у каждого нового кандидата.
- **Ошибки в квалификации**: Бот четко отсекает людей младше 14 лет.
- **Медленный ответ**: Бот отвечает мгновенно 24/7, не давая кандидату «остыть».
- **Сложность в обработке возражений**: AI-интеграция (GigaChat) позволяет мягко отрабатывать сомнения («пирамида», «сетевой») без участия человека.

## 🛠 Решение (MVP)
Бот представляет собой интерактивную воронку:
1. **Квалификация**: Сбор данных (пол, имя, стиль общения, возраст).
2. **AI-анализ**: GigaChat анализирует свободный рассказ пользователя о себе и извлекает город, профессию и хобби.
3. **Прогрев**: Демонстрация истории успеха наставника и короткие видео о компании/продукте.
4. **Обработка вопросов**: Интеллектуальный чат с бизнес-консультантом на базе AI.
5. **Конверсия**: Выдача персональной реферальной ссылки на регистрацию.

## 💻 Технологический стек
- **Язык**: Python 3.11+
- **Библиотека бота**: aiogram 3.x (асинхронность)
- **AI-мозг**: GigaChat API (через aiohttp)
- **Хранение состояний**: MemoryStorage (FSM)
- **Конфигурация**: python-dotenv

## 📈 Метрики успеха
- % пользователей, дошедших от `/start` до ссылки на регистрацию.
- Точность извлечения данных AI из свободного текста.
- Количество успешных ответов на возражения.
